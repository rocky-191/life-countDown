# 人生倒计时 (life-countdown)

一个简洁美观的人生剩余时间计算器，帮助你正视时间的珍贵。

## 功能特性

- **输入年龄**：填入当前年龄和预计寿命
- **实时计算**：自动计算剩余的年、天、小时、分钟
- **进度条展示**：直观显示人生已度过百分比
- **随机名言**：页面底部每天刷新不同的珍惜时间/生命名言

## 使用方法

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

### 本地运行

```bash
# 方式一：直接打开
open index.html

# 方式二：使用 Python 临时服务器
python3 -m http.server 3000
# 然后访问 http://localhost:3000
```

## 技术栈

- 纯原生 HTML + CSS + JavaScript
- 无框架依赖，单文件部署
- 响应式设计，支持手机和电脑

## 名言来源

精选 20 条古今中外的珍惜时间与生命名言，每次刷新页面随机展示一条。
